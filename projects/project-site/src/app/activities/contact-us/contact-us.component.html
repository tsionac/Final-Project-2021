Who we are

Our customer has web pages where his employees can edit/ view entities.
The services we provide are: a widget that can be embedded on the customer’s web pages that will log who edits/ views an entity.
The users that edit/ view an entity will be able to see which other users are editing/ viewing the same entity at real time.
In addition, we will provide a web interface that a higher level user (eg. Manager user) will be able to login and see information about users activity for each existing entity on the customer's web page (eg. who edit/ view, when)

Vision

Create a dashboard that will show information about the viewed/edited entities in customer's systems. We will offer an injectable code that can be placed within existing website entities. This code will send information about the entities’ viewers and editors (of each entity that has the injectable code). At start we will support Angular.js components with the option to extend to more frameworks and extend the offered component library we provide to our users.
